References

What we looked at. What we took.

The Row
Primary reference — entire aesthetic

Maximum white space

Serif at weight 200–300 only

No decorative elements

No hero images — text is the hero

Pricing shown plainly, no hype

Lemaire
Tone and spacing reference

Generous line height in body

Understated navigation style

No accent colour anywhere

Hierarchy via size, not colour

Body copy as a luxury signal

Toteme
Grid and layout reference

Strict column structure

Horizontal rules as dividers

Labels in small caps with tracking

Consistent section-to-section rhythm

Photography secondary to type

Jil Sander
Copywriting register

Short, declarative sentences

No marketing superlatives

Trust implied, not stated

Silence as a design element

Nothing explained twice
