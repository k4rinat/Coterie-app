Design Intent

Why dark mode was worth designing properly.

Not an afterthought

Dark mode was planned from the very start. The CSS token system was built to support it before a single dark colour was chosen. This meant it wasn't retrofitted — it was designed in from day one.

The mood shifts

The light version feels editorial — like a magazine. The dark version feels more like a private members' club at night. Both are valid. Both serve the same content. The typography holds in both.

Warm not cold

The dark backgrounds have a very slight warm undertone — #0c0c0c rather than #000000. Pure black reads as harsh on screen. The slight warmth keeps the page feeling considered, not stark.

Same type, different atmosphere

Cormorant Garamond at weight 300 works differently in each mode. In light, it reads as crisp and editorial. In dark, it reads as more intimate and considered. The same font does different work.
