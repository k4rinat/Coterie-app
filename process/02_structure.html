Navigation

Navigation decisions made before any layout.

Coterie
How It Works Pricing FAQ For Brands
Apply for access →
Wordmark placement Left-aligned. Not centred. Centred logos read as fashion editorial — this is a product, not a magazine.
Link style Inter 300 Light, no underline, no hover colour change. The restraint is intentional — the nav shouldn't compete with the page.
CTA in nav Right-aligned. Always visible. Not a button — a hairline text link with an arrow. Same as every other CTA on the site.
Home not in nav Wordmark is the home link. Adding a separate "Home" item wastes a slot and adds no value.

Hero

Full viewport height. The only content is a headline and one CTA. No image, no subheading, no bullets.

Decision: If the headline is strong enough, it doesn't need support. Make the headline strong enough.

Pillars

Three value propositions in a horizontal row. No heading above them — they speak for themselves.

Decision: No background change between hero and pillars. One continuous page.

Disciplines

All the creative roles on the platform listed out. Two columns. Ruled rows. No icons.

Decision: This section is a signal — seeing your job title in the list tells you immediately this is for you.

Membership

Two tier cards side by side. Border only — no fills, no highlighted "recommended" tier.

Decision: Coterie doesn't push. Both tiers presented as equals. The user decides.

Editorial Quote

One line. Cormorant italic. Centred. No attribution needed.

Decision: A moment of pause before the final push. Like a pull quote in a magazine.

Final CTA

Identical to the hero CTA. Same words, same style. The page is a loop.

Decision: If someone scrolls the whole page and still wants to apply, they shouldn't have to scroll back up.
